# QC App - Marketing Storyboard

This folder contains the marketing storyboard for the QC (Quality Control) relationship check-in app.

## Files

- **storyboard.html** - Self-contained HTML page showcasing all major features
- **screenshots/** - Mobile viewport screenshots (390x844px - iPhone 14 Pro size)

## Storyboard Structure

The storyboard follows the user journey through three acts:

### Act 1: Getting Started (Screenshots 1-3)
1. **Landing Page** - Hero section with value proposition
2. **Onboarding Welcome** - Quick 2-minute personalization
3. **Onboarding Quiz** - Communication style discovery

### Act 2: Core Features (Screenshots 4-11)
4. **Dashboard** - Relationship command center with activity feed
5. **Check-in Start** - Category selection with session rules
6. **Check-in Discussion** - Guided prompts for conversations
7. **Notes Library** - Three-tier privacy system (shared/private/draft)
8. **Growth Gallery** - Milestone tracking and progress visualization
9. **Love Languages** - Actionable love language management
10. **Reminders** - Smart reminder system for connection
11. **Requests** - Partner communication and request management

### Act 3: Customization (Screenshots 12-13)
12. **Session Settings** - Customizable check-in duration and rules
13. **Categories Manager** - Custom discussion topics and prompts

## Key Features Highlighted

- **Three-tier privacy system** (private/shared/draft notes)
- **Six-step guided check-in flow** with session bookends
- **Mobile-first design** with gesture support
- **Progress tracking** with milestones and analytics
- **Customizable session rules** (timeouts, turn-based, duration)
- **Love languages integration** with actionable examples

## Viewing the Storyboard

Simply open `storyboard.html` in any modern web browser:

```bash
open storyboard.html
```

Or drag the file into your browser.

- Emailed directly

- Opened locally without any dependencies
